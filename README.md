# Bank Churn Prediction Project
This project predicts whether a customer will leave the bank or not.

## Features
- Machine Learning models: Random Forest, Logistic Regression, KNN + XGBoost
- Achieved 92% accuracy
- Flask web application for prediction

## Technologies Used
- Python
- Flask
- Machine Learning
- Pandas, NumPy

## Description
I developed a machine learning model and deployed it using a Flask web application to provide real-time predictions.
